# 花生壳内网穿透配置指南

> 让外网安全访问本地 OpenClaw (http://127.0.0.1:18789)

## 🎯 目标

- 外网通过**花生壳赠送的免费域名**访问本地 OpenClaw
- 不直接暴露 18789 端口
- 使用 HTTPS 加密传输

**无需购买域名！** 花生壳免费赠送 `xxx.hsk.oray.com` 三级域名

---

## 📋 前置要求

1. **花生壳账号**：注册 Oray 账号（https://hsk.oray.com）
2. **实名认证**：完成个人/企业实名认证
3. **购买服务**：
   - 免费版：1 个映射，1Mbps 带宽
   - 专业版：推荐，支持 HTTPS、多映射

## 🚀 安装花生壳客户端

### macOS 安装

```bash
# 1. 下载客户端
curl -o hsk.dmg https://dl.oray.com/hsk/mac/phddns.dmg

# 2. 安装后启动，或使用命令行版本
brew install --cask phddns
```

### Linux 安装

```bash
# 下载并安装
wget https://dl.oray.com/hsk/linux/phddns_5.3.0_amd64.deb
sudo dpkg -i phddns_5.3.0_amd64.deb

# 启动服务
sudo phddns start
```

### Windows 安装

直接下载安装包运行：https://dl.oray.com/hsk/windows/phddns.exe

## ⚙️ 配置内网穿透

### 步骤 1：登录花生壳客户端

```bash
# 命令行登录（Linux/macOS）
phddns login -u 你的账号 -p 你的密码

# 或打开 GUI 客户端扫码登录
```

### 步骤 2：创建映射规则（使用免费域名）

**花生壳会免费赠送一个三级域名，无需购买！**

1. 访问 https://console.hsk.oray.com
2. 点击「内网穿透」→「添加映射」
3. 填写映射信息：

```
应用名称：OpenClaw Gateway
映射类型：HTTPS（推荐）或 HTTP
外网域名：xxx.hsk.oray.com  ← 花生壳免费赠送的域名
外网端口：443（HTTPS）或 80（HTTP）
内网主机：127.0.0.1
内网端口：18789
```

**免费域名格式**：`你的用户名.hsk.oray.com` 或系统分配的随机域名

### 步骤 3：安全加固（重要！）

#### 1. 设置访问密码

在花生壳控制台开启「访问控制」：
```
基础认证：启用
用户名：admin（或自定义）
密码：强密码（16位以上，含大小写+数字+符号）
```

#### 2. 限制访问 IP（可选）

```
IP 白名单：你的手机/笔记本公网 IP
```

#### 3. OpenClaw 端配置

编辑 `~/.openclaw/openclaw.json`：

```json
{
  "gateway": {
    "port": 18789,
    "host": "127.0.0.1",
    "security": {
      "allowRemote": false,
      "requireAuth": true
    }
  }
}
```

**注意：** `allowRemote: false` 确保只能通过花生壳代理访问，不能直接访问 18789 端口。

## 🔒 防火墙配置

### macOS

```bash
# 阻止外网直接访问 18789
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/local/bin/openclaw
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --block /usr/local/bin/openclaw

# 或使用 pfctl（高级）
echo "block in proto tcp from any to any port 18789" | sudo pfctl -ef -
```

### Linux (iptables)

```bash
# 只允许本地访问 18789
sudo iptables -A INPUT -p tcp --dport 18789 -s 127.0.0.1 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 18789 -j DROP

# 保存规则
sudo iptables-save > /etc/iptables/rules.v4
```

### Linux (ufw - 推荐)

```bash
# 拒绝外部访问 18789
sudo ufw deny 18789/tcp

# 允许本地回环
sudo ufw allow from 127.0.0.1 to any port 18789
```

## 🚨 安全提醒

1. **定期更换密码**：建议每 3 个月更换一次访问密码
2. **监控访问日志**：定期检查花生壳控制台的访问记录
3. **启用双因素认证**：Oray 账号开启 2FA
4. **限制访问时间**：如需可设置花生壳的时段访问控制
5. **使用 HTTPS**：强制加密传输，防止中间人攻击

## 🔧 故障排查

### 问题 1：无法连接

```bash
# 检查花生壳客户端状态
phddns status

# 检查 OpenClaw 是否运行
curl http://127.0.0.1:18789/health

# 重启花生壳
phddns restart
```

### 问题 2：连接超时

- 检查本地防火墙是否放行花生壳客户端
- 确认 OpenClaw 监听的确实是 127.0.0.1:18789
- 查看花生壳客户端日志：`tail -f /var/log/phddns.log`

### 问题 3：HTTPS 证书错误

- 确保花生壳映射类型选择 HTTPS
- 免费版花生壳提供自动 SSL 证书
- 如需自定义域名证书，升级到专业版
